Boost the Website with G Map image lazy loading,

== Description ==

Today speed optimization is big issue for bloggers and business websites, Most of the websites use the Google Map in footer, Google give the reward to high-speed websites but Google map makes site slow, With this plugin load the Google map when scrolling down.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress3. Use the short code. [gmap lat="34.184756" long="-77.932323"]
4s. Enjoy
